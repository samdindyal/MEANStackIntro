In order to use the MEAN Stack, you need to have two things installed:

  • Node.js - https://nodejs.org/en/
  • MongoDB - https://www.mongodb.org
These can easily be found on their respective websites for any major platform. Node.js has an installer while MongoDB requres some terminal work. Below is a screenshot of what the installer on Mac OS X looks like.

On Mac OS X, MongoDB can be installed via Homebrew:

   $ brew install mongodb

The instructions for installing MongoDB on other platforms can be found in the link below:

https://www.mongodb.org/downloads#production


Once Node.js is installed, the syntax for running your Node.js program is as follows:

   $ node script.js
