The MEAN Stack is a full stack software bundle built on JavaScript. MEAN stands for mongoDB, express.js, AngularJS and node.js.

mongoDB is a non-relational, or NoSQL database. It stores data in BSON, or, Binary JavaScript Object Notation (JSON).

express.js is a web application server framework built on node.js. It contains objects which helps with organizing the server side code, written with node.js.

AngularJS is a structural framework used for creating dynamic webapps. It uses databinding and dependency injection to make code cleaner and more concise.

node.js is a cross-platform runtime environment for developing server side web applications. It is built on JavaScript and has only recently gotten its first stable release.


The MEAN Stack was initially released on February 12th, 2014 and its first stable release was on October 7th, 2014.

When using the MEAN Stack, mongoDB is typically used as a database, express.js and node.js are used for running the web server and serving content and other files and AngularJS is used to provide a more dynamic user experience with the use of its two-way data binding and more.
